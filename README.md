# TypeScript Project

A simple TypeScript project using pnpm and tsx.

## Prerequisites

-   Node.js
-   pnpm

## Setup

1. Install dependencies:

```bash
pnpm install
```

2. Development:

```bash
pnpm dev
```

3. Build:

```bash
pnpm build
```

4. Run:

```bash
pnpm start
```

## Project Structure

-   `src/` - Source code
-   `dist/` - Compiled JavaScript (generated after build)
